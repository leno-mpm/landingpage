# Al Día ☀️

**Al Día** es una **landing page interactiva** creada con **TailwindCSS** y **JavaScript**, cuyo objetivo es presentar una aplicación que ayude a los usuarios a **gestionar y controlar sus suscripciones digitales** de forma fácil y organizada.

La aplicación busca **solucionar los principales dolores** de los jóvenes y adultos que manejan múltiples servicios como **Netflix, Spotify, Disney+ o iCloud**, ayudándoles a:

- Recordar las fechas de cobro de cada servicio.  
- Tener claridad sobre los gastos mensuales.  
- Identificar suscripciones activas o innecesarias.  
- Evitar cobros automáticos inesperados.  

Con una **interfaz moderna, clara y atractiva**, **Al día** ofrece una experiencia sencilla que promueve el **orden financiero** y el **control personal** sobre los pagos digitales.
